#  Manual de Buenas Prácticas para Trabajos Universitarios en Grupo

Este repositorio contiene un manual colaborativo que recopila buenas prácticas para organizar, planificar y elaborar trabajos en equipo a nivel universitario. El objetivo es aprender a usar Git y GitHub como herramientas de control de versiones, trabajo colaborativo y documentación profesional.

---

##  Objetivos del proyecto
- Fomentar el uso de Git y GitHub en proyectos no técnicos.
- Desarrollar habilidades de colaboración y revisión de documentos.
- Crear y mantener un manual con buenas prácticas de trabajo en equipo.
